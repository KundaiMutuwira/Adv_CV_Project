# Adv_CV_Project
People in Social Context - Visual Social Relationship Recognition

Understanding social relationships in images is essential for socially-aware Al, with applications in human-robot interaction and scene understanding. Given an image with multiple people, the task is to infer their relationships (e.g., friends, colleagues, family) using only visual and contextual cues, without identity information. This requires reasoning about interactions and implicit social signals rather than isolated individuals. Prior work has explored graph-based [1] and context-aware approaches [2]. Building on these, this project aims to investigate complementary methods inspired by multimodal social intelligence frameworks (e.g. [3]), moving toward richer and more robust modeling of human relationships, as highlighted in [5]. Datasets [4] and [5] are publicly available.
